# Airbnb Clone

### Developed a feature-rich Airbnb Clone, a full-stack web application that emulates the core functionalities of the popular Airbnb platform. 

## Tech Stack

Next.js<br/> React<br/> Tailwind CSS<br/>
Node.js<br/> Express.js<br/>
MongoDB<br/>

## Get started

### Install packages
npm install

### Configure .env file
DATABASE_URL=<br/>
GOOGLE_CLIENT_ID=<br/>
GOOGLE_CLIENT_SECRET=<br/>
GITHUB_ID=<br/>
GITHUB_SECRET=<br/>
NEXTAUTH_SECRET=

### Setup Prisma
npx prisma db push

### Start the app
npm run dev
